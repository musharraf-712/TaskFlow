# TaskFlow 🗂️

A role-based task management system where admins assign tasks and employees track and update their progress.

## Features
- Admin & Employee login system
- Admin can assign tasks to employees
- Employees can view assigned tasks
- Update task status (Pending, Accepted, Completed, Failed)
- Track task progress easily
- Data stored using localStorage 

## Tech Stack
- React
- Tailwind CSS
- JavaScript
- localStorage
